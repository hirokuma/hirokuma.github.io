全然Rustの勉強が進まない。
今日は `Error` 関連だ。

* [rust: thiserror で自作のエラー型を作る - hiro99ma blog](https://blog.hirokuma.work/2025/12/20251214-rst.html)
* [rust: anyhow::Resultからの脱却 - hiro99ma blog](https://blog.hirokuma.work/2026/02/20260217-rst.html)

`thiserror` を使わず、`anyhow` も使わず、
例えるなら自作ライブラリ内で使用したクレートのそれぞれがエラーを返したけど呼び出し元には自作ライブラリのエラーとして返す、
そんなものをイメージした。

まず思ったのは、Go言語の `fmt.Errorf()` みたいな「これはエラーですよ」という扱いをする簡単なメソッドや `struct` はなさそうということだ。
既存で定義してあるエラーを借りてくるか、`std::error::Error` トレイトを使って自作するようだ(`thiserror` などを使わない場合は)。  
`anyhow::bail!()` で楽してエラーを作ってきたのを反省する次第である。。。

`From` トレイトを書いておくと `?` があると自動的に呼び出されてよいと書いてあったのだが、明示したほうがわかりやすくないかい？ 
いや、それは今回は自分で全部作っていて、なおかつエラーしか返さない実装をしているからだろう。  
・・・本当にそうなんだろうか。今回だと `MyU64Error` がなんで `MyError` になるのか疑問に思ってしまいそうだ。
`Result<>` を返すメソッドで `?` が使われてコンパイルエラーになってないならどこかに `From` トレイトの実装があるんだろう、と覚えておくしか無いのか。

あとは `Error` トレイトの `source()` だが、引数が `&self` で戻り値が `Option<>` の `&(dyn Error + 'static)` なので、
今回作った `enum MyError` の列挙子(英語だとvariant)は `Error` トレイトを実装した型じゃないと `Some()` で囲めなさそうだった。
既に定義してある固定値を返すなら `Some(&なんちゃら)` でできそうだけど、`match` で各アームにした先で `&u64` とかもらってもどうしようもできず `None` を返すことしかできないと思う。

まだ学習が足りんな。

```rust
#[derive(Debug)]
pub struct MyU64Error(u64);

impl std::fmt::Display for MyU64Error {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        write!(f, "My u64 error({})", self.0)
    }
}

impl From<u64> for MyU64Error {
    fn from(value: u64) -> Self {
        Self(value)
    }
}

impl std::error::Error for MyU64Error {}

#[derive(Debug)]
pub struct MyStringError(String);

impl std::fmt::Display for MyStringError {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        write!(f, "My String error({})", self.0)
    }
}

impl From<String> for MyStringError {
    fn from(value: String) -> Self {
        Self(value)
    }
}

impl std::error::Error for MyStringError {}

#[derive(Debug)]
pub enum MyError {
    ErrA(MyU64Error),
    ErrB(MyStringError),
    ErrC(std::io::Error),
}

impl std::fmt::Display for MyError {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self {
            MyError::ErrA(e) => write!(f, "Error A({})", e),
            MyError::ErrB(e) => write!(f, "Error B({})", e),
            MyError::ErrC(e) => write!(f, "Error C({})", e),
        }
    }
}

impl From<MyU64Error> for MyError {
    fn from(value: MyU64Error) -> Self {
        Self::ErrA(value)
    }
}

impl From<MyStringError> for MyError {
    fn from(value: MyStringError) -> Self {
        Self::ErrB(value)
    }
}

impl From<std::io::Error> for MyError {
    fn from(value: std::io::Error) -> Self {
        Self::ErrC(value)
    }
}

impl std::error::Error for MyError {
    fn source(&self) -> Option<&(dyn std::error::Error + 'static)> {
        match self {
            MyError::ErrA(e) => Some(e),
            MyError::ErrB(e) => Some(e),
            MyError::ErrC(_e) => None,
        }
    }
}

fn main() {
    match return_err_a() {
        Ok(v) => println!("This value is {}.", v),
        Err(e) => eprintln!("Error; {}", e),
    }
    match return_my_err() {
        Ok(v) => println!("This value is {}.", v),
        Err(e) => eprintln!("Error; {}", e),
    }
}

fn return_err_a() -> Result<u64, MyU64Error> {
    // Err(MyU64Error(12345))
    Err(12345)?
}

fn return_my_err() -> Result<u64, MyError> {
    // Err(MyError::ErrA(return_err_a().unwrap_err()))
    Ok(return_err_a()?)
}
```
