## jq -s

`jq` コマンドでは `-s`, `--slurp` オプションを付けると扱いやすくなる。

例えばこういう値が環境変数 `list` に入っていたとしよう。

```json
{"address":"bcrt1p89pfj7urvmfl7h43s2acqljgzwk4ms0qdw8z0h92xzatrtah9f3sfcz8fe","confs":8,"out_point":"9946804160d3457c4a2d3ae24161b098f18120aabb9544c509865175397e9797:0","value":108571}
{"address":"bcrt1ptd2ty3j7safnkkl47rqn2gfd2jgkza3q7ju7f40pt6s7vgs54ymseq2vz4","confs":189,"out_point":"66203f09af028528da19e4ec7754bdbf3a30cb03cf961d2380646cef5b718cf2:2","value":10000}
{"address":"bcrt1p5mk90m0n3pavy7zm9e23wf6ag67yp43y3ud024hcvf0py4gw5tqsx29ldt","confs":189,"out_point":"a629d4654b1c23accbdb4f23bdddd1aa53971a1d4a15771cf07d96a80aab94c6:0","value":10000}
```

データ数を求めるならこんな感じ。

```bash
$ echo $list | jq -s 'length'
3
```

`"value"` だけ抽出するならこれだけでよい。

```bash
$ echo $list | jq '.value'
108571
10000
10000
```

ただこれだと `jq` のコマンドがうまく使えないので、こんな感じで配列にするのが良さそう。
そうすると `add` で合計することができる。

```bash
$ echo $list | jq -s 'map(.value)'
[
  108571,
  10000,
  10000
]

$ echo $list | jq -s 'map(.value) | add'
128571
```
